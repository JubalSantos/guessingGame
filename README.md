# The Guessing Game

The game has two players: the oracle, who knows a secret number and the player has to guess that number.
The player only gets one guess.

```
Example:
Oracle: What is your name?
Player: Bob
Oracle: What is the magic number, Bob?
Player: 123
Oracle: TOO LOW
```
